# Java Concepts Used

## Variables
Variables are used to store data. In this project:
- String name stores the user's name.
- int age stores the user's age.

## Scanner
The Scanner class is used to read input from the keyboard.

Example:
Scanner scanner = new Scanner(System.in);

## Input and Output
System.out.print() and System.out.println() are used to display messages to the user.

Scanner methods such as nextLine() and nextInt() are used to receive user input.

## Main Method
The main() method is the entry point of the Java program.

# PersonalizedGreeting

A simple Java console application that accepts a user's name and age and displays a personalized greeting.

## How to Run

1. Open the project in IntelliJ IDEA.
2. Run `PersonalizedGreeting.java`.
3. Enter your name and age.

## Sample Input

Enter your name: Harsha
Enter your age: 33

## Sample Output

Hello Harsha, you are 33 years old.

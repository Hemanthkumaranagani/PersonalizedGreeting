# PersonalizedGreeting

A simple Java console application that accepts a user's name and age and displays a personalized greeting.

## Features

- Accepts user input using Scanner
- Demonstrates variables and data types
- Uses console input/output
- Displays a personalized greeting message

## Requirements

- Java JDK 8 or higher

## Compilation

```bash
javac src/com/greet/PersonalizedGreeting.java
